# file-type-cli

> Detect the file type of a file or stdin

## Install

```
$ npm install --global file-type-cli
```

## Usage

```
$ file-type --help

  Usage
    $ file-type <filename>
    $ file-type < <filename>

  Example
    $ file-type unicorn.png
    png
    image/png
```

[Supported file types](https://github.com/sindresorhus/file-type#supported-file-types)
